How to Understand Tennis!

  1. Understanding the score
    • Matches
      • Each match consists of 3-5 sets
      • First player to win 3 sets wins the match
    • Sets
      • The first player to score 6 games wins the set
      • Sets are win by 2 games, capped at 7 games
      • If a set reaches 6-6, a tiebreaker is played
      • The tiebreaker is first to 7 points, win by 2
    • Games
      • Each game is to 4 points, win by 2
    • Scoring within games
      • 0 points = "love"
      • 1 point = "fifteen"
      • 2 points = "thirty"
      • 3 points = "forty"
      • If the score is 3-3, then it is "deuce"
      • If the score is 4-3, and the server is winning, it is "advantage in"
      • If the score is 3-4, and the returner is winning, it is "advantage out"
  2. Know the lines
    • The alley
      • Use the farthest outside alley when playing doubles
      • Use the inside alley when playing singles
    • The service box
      • When serving from the left side, your serve must land in the right service box
      • When serving from the right side, your serve must land in the left service box
    • If any part of the ball touches the line, that shot is in
